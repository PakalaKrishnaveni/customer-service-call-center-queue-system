# Customer Service Call Center Queue System

## 📌 Project Description

The **Customer Service Call Center Queue System** is a Java-based console application designed to efficiently manage customer queries in a call center environment.
It uses a queue mechanism to handle customer requests, ensuring smooth and organized processing.

## 🚀 Features

* ➤ Add Normal Customers (added to end of queue)
* ➤ Add VIP Customers (high priority – added to front)
* ➤ Serve Customers (First-In-First-Out order)
* ➤ Display All Customer Records
* ➤ Timestamp for each customer query
* ➤ Email & Mobile Number Validation

## 🛠️ Technologies Used

* Java
* Collections Framework (ArrayDeque)
* Object-Oriented Programming (OOP)
* Java Date & Time API

## 🧠 Concepts Used

* Queue (FIFO Principle)
* Deque (Double-Ended Queue)
* Encapsulation
* Layered Architecture

## 📂 Project Structure

* `com.base` → Customer class
* `com.business_layer` → Business logic
* `com.service_layer` → Main execution
* `com.data_validation` → Validation logic

## ▶️ How to Run

1. Clone the repository
2. Open the project in Eclipse / IntelliJ IDEA
3. Navigate to `CustomerMain.java`
4. Run the program
